# mirrous-nextjs-starter

Next.js를 프로젝트에서 사용하려 할 때, 사용하기 쉽도록 만든 보일러플레이트 저장소입니다.

## Installation

패키지 매니저로는 yarn을 사용하고 있습니다.  
yarn이 없는 경우 설치해주세요

```bash
brew install yarn
```

패키지 설치

```bash
yarn
```

## Tech Stack

- React
- TypeScript
- Next.js
- styled-components
- eslint
- stylelint
